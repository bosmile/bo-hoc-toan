# 📊 QUẢN LÝ HÓA ĐƠN MUA HÀNG

Ứng dụng web quản lý hóa đơn mua hàng với đầy đủ tính năng CRUD.

## 🚀 Tính Năng Chính

### 1. Quản lý hóa đơn
- **Thêm hóa đơn mới** với nhiều sản phẩm
- **Phân loại**: BIGC, SP/LZD, Khác
- **Thông tin đầy đủ**: Tên SP, giá, số lượng, thành tiền, ngày hóa đơn
- **Upload ảnh hóa đơn** (tùy chọn)

### 2. Xem và tìm kiếm
- **Danh sách hóa đơn** theo ngày
- **Tìm kiếm** theo cửa hàng, ghi chú
- **Lọc** theo phân loại, thời gian
- **Sắp xếp** theo nhiều tiêu chí

### 3. Xuất báo cáo
- **Xuất file Excel** toàn bộ dữ liệu
- Định dạng tiền tệ VND
- Cột thông tin đầy đủ

### 4. Lưu trữ
- **LocalStorage**: Lưu trữ trên trình duyệt
- **Không cần backend**: Hoạt động offline
- **Dữ liệu mẫu**: Có sẵn dữ liệu demo
