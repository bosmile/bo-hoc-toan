# **App Name**: BƠ HỌC TOÁN

## Core Features:

- Interactive Math Problem Generator: A tool to generate mathematical problems of the form 'A ± B ± C = D'. Users can specify which variable (A, B, C, or D) is the unknown and set Min/Max ranges for the values. The system intelligently ensures all intermediate calculations and the final result are non-negative.
- Chapter Navigation Sidebar: A fixed sidebar offering structured navigation, separating 'Toán Archimedes' and 'Toán tư duy Singapore' into distinct groups, each containing dedicated links for 'Chuyên đề 1', 'Chuyên đề 2', and 'Chuyên đề 3'.
- Chapter 1 Configuration Page: A detailed configuration interface for 'Chuyên đề 1 - Toán Archimedes', allowing users to select the unknown variable and precisely adjust the minimum and maximum value ranges for the numbers involved in the generated problems.
- Combined Practice Test Builder: On the homepage, users can assemble custom practice tests by selecting the desired number of questions from various specialized chapters (e.g., 5 questions from Chapter 1 + 15 questions from Chapter 2).
- Quick Configuration Access: The homepage provides convenient quick-access options and settings for each chapter, enabling swift adjustments to problem difficulty and common parameters without navigating to individual chapter configuration pages.
- Printable PDF Export: Ability to export generated problem sets to an A4 PDF format, featuring carefully calibrated margins to ensure perfect printability for children's worksheets.

## Style Guidelines:

- The concept of 'BƠ HỌC TOÁN' (avocado/butter and learning math) inspires a natural, calming, and focused aesthetic suitable for an educational app. A light color scheme will promote clarity and readability. The primary color is a soft, fresh green, evoking growth and a natural learning environment.
- Primary color: `#559933` (HSL: 105, 50%, 40%) - a balanced, fresh green for key interactive elements.
- Background color: `#EEF3E9` (HSL: 105, 20%, 95%) - an exceptionally light green-tinted off-white, offering a clean, airy, and gentle canvas.
- Accent color: `#BDD12E` (HSL: 75, 70%, 60%) - a vibrant yellow-green to highlight important actions, calls-to-action, and add a cheerful accent to the interface.
- Headline and body font: 'Inter' (sans-serif) for its modern, clean, and highly legible characteristics, ensuring mathematical problems and instructions are easy to read for learners.
- Consistent usage of Lucide Icons across the application to provide clear, vector-based, and modern visual cues that enhance usability.
- A consistent two-column layout featuring a static navigation sidebar on the left for easy access to chapters and a dynamic main content area that updates based on selection, as requested by the user.
- Incorporate subtle and gentle animations for UI transitions, element states, and feedback mechanisms to create a responsive and engaging user experience for children without being distracting.
